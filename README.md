# Number Guessing Game

This project is an interactive number guessing game developed using JavaScript. It was initially created as part of a programming course and later improved with additional features and enhancements.

## About the Project

The objective of the game is to guess a randomly generated number within a defined range. The player receives hints after each attempt and must find the correct number within a limited number of tries.

## Features

* Random number generation without repetition
* Input validation
* Hint system (higher or lower)
* Limited number of attempts
* Difficulty levels:

  * Easy (1–10)
  * Medium (1–50)
  * Hard (1–100)
* Progressive difficulty unlocking
* Victory counter by difficulty
* Input blocking after game ends
* Game reset functionality
* Dynamic feedback messages

## Technologies Used

## Concepts Applied
